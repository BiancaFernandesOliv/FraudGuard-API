# 🛡️ FraudGuard API

API desenvolvida em C# e ASP.NET Core para análise de transações suspeitas, inspirada em cenários reais de prevenção à fraude em fintechs.

## 📌 Sobre o projeto

O FraudGuard nasceu como um projeto de estudos para consolidar conhecimentos em desenvolvimento backend com .NET, aplicando conceitos de Programação Orientada a Objetos, boas práticas de desenvolvimento e análise de regras de negócio.

A proposta é simular um sistema capaz de identificar transações potencialmente suspeitas, classificando riscos e permitindo a evolução gradual para cenários mais complexos de detecção de fraude.

## 🎯 Objetivos

* Praticar C# e ASP.NET Core;
* Aplicar conceitos de Programação Orientada a Objetos;
* Implementar regras de detecção de fraude;
* Evoluir a arquitetura do projeto gradualmente;
* Simular cenários reais inspirados em prevenção à fraude.
